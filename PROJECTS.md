# Темы семестровых проектов

Ниже — 30 возможных тем для проекта второго полугодия.
Формулировки ниже даны с позиции manager/product-owner.
Задача — на основе этого сделать анализ, UX-концепцию и реализовать решение.

**Важно:** Все темы уникальны и не пересекаются с темами первого полугодия.

---

## 1. Система управления складом

**Описание проекта (от бизнеса)**
Приложение для учёта товаров на складе, перемещений между зонами и контроля остатков.

**Бизнесовые требования (верхнеуровневые)**
- Пользователь может добавлять товары с параметрами (название, артикул, категория).
- Пользователь может создавать зоны хранения на складе.
- Пользователь может фиксировать перемещения товаров между зонами.
- Пользователь ожидает видеть текущие остатки по каждой зоне.
- Пользователь может устанавливать минимальные пороги остатков.
- Пользователь ожидает уведомления о товарах ниже порога.
- Пользователь может проводить инвентаризацию по зоне.
- Пользователь ожидает историю всех перемещений по товару.
- Пользователь может формировать отчёт по оборачиваемости товаров.
- Требуется авторизация и разделение данных между организациями.
